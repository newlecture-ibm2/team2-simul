# FitCheck MVP 요구사항 기능서

> **입어보고, 모으고, 공유하다**
> AI 가상시착 결과를 중심으로 개인 옷장을 구성하고 커뮤니티에 공유하는 패션 소셜 플랫폼

---

## 서비스 개요

| 항목 | 내용 |
|------|------|
| 타겟 사용자 | 패션에 관심 많은 20–35세 일반 소비자 |
| 플랫폼 | iOS / Android (모바일 우선) |
| 개발 범위 | MVP — 핵심 4기능 중심 |

**서비스 설명**
패션에 관심 많은 일반 소비자가 실제 구매 전 AI로 옷을 시각적으로 경험하고, 시착 결과를 커뮤니티에 공유하며, 관심 아이템을 개인 옷장에 모아두고 외부 쇼핑몰로 바로 연결되는 End-to-End 패션 디스커버리 서비스.

### 기능 우선순위

| 순위 | 기능 | 중요도 |
|------|------|--------|
| 1 | AI 가상시착 (Virtual Try-On) | P1 — 핵심 차별점 |
| 2 | 개인 옷장 (디지털 클로젯) | P2 — 핵심 리텐션 |
| 3 | 커뮤니티 피드 + 시착 결과 공유 | P3 — 성장 엔진 |
| 4 | 제품 태그 & 쇼핑 연결 | P4 — 수익화 기반 |

---

## 1. AI 가상시착 `P1`

### 목표
사용자가 자신의 사진에 원하는 옷을 AI로 입혀보고 결과를 즉시 확인·저장·공유할 수 있다.

### 사용자 플로우

**Step 1 — 사진 입력**
- 갤러리에서 본인 전신/반신 사진 선택 또는 카메라 촬영
- 얼굴 인식 불필요 (신체 실루엣 기반 처리)
- 권장 비율: 3:4 세로

**Step 2 — 옷 선택** (3가지 경로)
- ① 커뮤니티 피드 게시물의 태그 상품 직접 선택
- ② 내 옷장에 저장된 아이템 선택
- ③ 외부 상품 URL 붙여넣기 (상품 이미지 크롤링)

**Step 3 — AI 생성**
- 서버 사이드 AI 모델이 합성 이미지 생성
- 처리 시간 목표: **10초 이내**
- 로딩 중 진행 애니메이션 표시
- 실패 시 재시도 1회 자동

**Step 4 — 결과 확인**
- 원본 사진 ↔ 시착 결과 스와이프 비교
- 결과 이미지 디바이스 저장
- 옷장에 아이템 추가 (원클릭)
- 커뮤니티 피드에 바로 공유

### 제약 조건
- 사용자 원본 사진은 시착 완료 후 서버에서 즉시 삭제 (개인정보 보호)
- 생성 결과물은 30일 보관 후 자동 삭제 (미저장 시)
- 1일 무료 시착 횟수: **5회** / 초과 시 추가 크레딧 안내

- ~~실시간 영상 시착 (카메라 스트리밍)~~
- ~~다중 아이템 동시 시착 (상하의 조합)~~

## 2. 개인 옷장 (디지털 클로젯) `P2`

### 목표
사용자가 관심 있는 옷을 디지털 옷장에 모아두고 카테고리별로 관리할 수 있다.

### 사용자 플로우

**Step 1 — 아이템 추가** (3가지 경로)
- ① AI 시착 결과에서 '옷장에 추가' 버튼
- ② 커뮤니티 피드 게시물의 태그 상품에서 저장
- ③ 외부 URL / 이미지 직접 추가

**Step 2 — 아이템 정보**
- 카테고리: 상의 / 하의 / 아우터 / 신발 / 액세서리
- 색상 태그 (자동 추출 + 수동 수정)
- 브랜드명 / 외부 쇼핑몰 링크 (선택)
- 개인 메모 (선택)

**Step 3 — 옷장 탐색**
- 카테고리 탭 필터
- 색상 팔레트 필터
- 그리드 뷰 (기본) / 리스트 뷰
- 최근 추가순 / 자주 시착순 정렬

**Step 4 — 아이템 활용**
- 해당 아이템으로 즉시 AI 시착 실행
- 외부 쇼핑몰 링크로 이동
- 아이템 삭제 / 편집

### 제약 조건
- MVP 아이템 저장 상한: 계정당 **200개**
- 아이템 이미지 크기: 최대 10MB
- 옷장은 기본 비공개 — 공개 전환 옵션 제공

- ~~코디 조합 저장 (룩북)~~
- ~~AI 기반 코디 추천~~
- ~~옷장 공유 / 팔로우~~

## 3. 커뮤니티 피드 + 시착 결과 공유 `P3`

### 목표
사용자들이 AI 시착 결과와 패션 사진을 피드에 공유하고 서로 반응할 수 있다.

### 사용자 플로우

**Step 1 — 게시물 작성**
- 이미지 1–5장 필수 (AI 시착 결과 또는 일반 패션 사진)
- 캡션 텍스트 (선택, 최대 300자)
- **제품 태그 1개 이상 필수** — 태그 없으면 업로드 불가
- 공개 / 비공개 설정

**Step 2 — 피드 탐색**
- 기본: 최신순 전체 피드 (MVP는 알고리즘 없이 시간순)
- 탭 전환: 전체 / 팔로잉
- 이미지 중심 그리드 레이아웃
- 무한 스크롤

**Step 3 — 게시물 상세**
- 이미지 풀뷰 스와이프
- 태그된 제품 목록 (탭하면 외부 링크 이동)
- 좋아요 / 댓글
- "이 옷으로 AI 시착 해보기" 버튼
- 게시자 프로필 / 팔로우 버튼

**Step 4 — 소셜 인터랙션**
- 좋아요 (하트)
- 댓글 (텍스트, 최대 200자)
- 팔로우 / 언팔로우
- 게시물 신고

### 제약 조건
- 제품 태그 없는 게시물 업로드 차단 (UX 가이드 문구 안내)
- 이미지당 최대 20MB, JPG/PNG/HEIC 지원
- 댓글은 최대 2depth (대댓글까지)

## 4. 제품 태그 & 쇼핑 연결 `P4`

### 목표
게시물 이미지 위에 제품을 태그하고 외부 쇼핑몰로 바로 연결한다.

### 사용자 플로우

**Step 1 — 태그 추가**
- 이미지 위 원하는 위치 탭 → 태그 핀 생성
- 브랜드명 / 제품명 텍스트 입력
- 외부 쇼핑몰 URL 입력 (필수)
- 이미지당 최대 **5개** 태그

**Step 2 — 태그 표시**
- 피드 그리드: 태그 존재 시 쇼핑백 아이콘 오버레이
- 상세 뷰: 이미지 탭 시 핀 표시 / 다시 탭 시 숨김 토글
- 하단 제품 리스트: 브랜드명 + 제품명 + 링크 버튼

**Step 3 — 쇼핑 연결**
- 태그 또는 리스트의 '구매하기' 버튼 탭
- 인앱 브라우저로 외부 쇼핑몰 URL 열기
- 뒤로가기 시 게시물 상세로 복귀

### 제약 조건
- URL 유효성 검사 필수 (http/https 형식)
- 태그는 게시물 필수 요소 — 없으면 업로드 불가
- MVP에서 URL 안전성 검사는 기본 블랙리스트 필터만 적용

## 5. 계정 & 프로필 (지원 기능)

### 목표
서비스 이용을 위한 최소한의 계정 기능과 개인 프로필 페이지를 제공한다.

### 회원가입 / 로그인
- 소셜 로그인: 카카오, 애플 (MVP 필수)
- 이메일 로그인 (선택)
- 비회원 피드 열람 가능 (시착·저장·공유는 로그인 필요)

### 프로필 페이지
- 프로필 이미지 + 닉네임 + 한줄 소개
- 내 게시물 그리드
- 팔로워 / 팔로잉 수
- 내 옷장 바로가기 버튼

### 설정
- 알림 ON/OFF (좋아요, 댓글, 팔로우)
- 계정 공개 / 비공개
- 로그아웃 / 회원탈퇴

## 6. 비기능 요구사항

### 성능
- 피드 로딩: 첫 콘텐츠 **2초 이내**
- AI 시착 생성: **10초 이내** (95th percentile)
- 이미지 Lazy Load 적용

### 보안 / 개인정보
- 사용자 원본 사진 시착 후 서버 즉시 삭제
- HTTPS 필수
- 개인정보처리방침 동의 온보딩 포함

### 접근성
- 이미지 alt 텍스트 필수
- 최소 터치 타겟 44×44pt
